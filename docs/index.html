PhotoAI Documentation

Project Overview

PhotoAI is an advanced image restoration system built on SDXL diffusion model architecture, providing significant quality improvements for photo-realistic image enhancement.

Key Features

  • Custom VAE architecture with enhanced reconstruction
  • Multiple model variants (Base, Lightning, Tiled, Face)
  • 4K/8K upscaling capabilities
  • Memory-optimized processing
